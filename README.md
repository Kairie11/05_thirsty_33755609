A Node.js web application for a premium beverage shop featuring multiple locations, customer registration, product search, and customer surveys.
Features

Home Page: Welcome page displaying the shop name and product categories
About Page: Information about the shop with a list of all store locations and their managers
Search Page: Search functionality for finding drinks by name and category
Registration Page: Customer registration form for membership
Survey Page: Customer feedback survey with demographic questions

The shop offers the following beverage categories:

Beer
Wine
Soft Drinks
Hot Drinks

CSS Features

Dark theme with gold/copper accent colors (#d4a574)
Gradient backgrounds for visual depth
Hover effects on navigation and interactive elements
Responsive form inputs with focus states
Card-based layouts for information display
Consistent styling across all pages

The customer survey collects:

First name and surname
Email address
Age
Preferred drink category (radio button selection)
Student status (checkbox)

Survey responses are displayed on a results page
